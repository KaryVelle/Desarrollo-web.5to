  4. Invierno

Lista Ordenada Revertida

Una lista con números ordenados revertidos requiere"ol reversed" y es una lista que, en vez de tener viñetas, tiene números.

  1. Primavera
  2. Verano
  3. Otoño
  4. Invierno

Lista Ordenada Iniciando Desde...

Una lista con números ordenados inicando desde un número determinado, requiere"ol start="n"" y es una lista que, en vez de tener viñetas, tiene números.

  3. Otoño
  4. Invierno
